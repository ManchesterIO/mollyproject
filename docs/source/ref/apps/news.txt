:mod:`molly.apps.feeds.news` -- News
====================================

.. module :: molly.apps.feeds.news

Used to display news feeds to the user.

Configuration
-------------

This module has no configuration in setup.py. Rather, feeds are controlled in
the database as discussed in :doc:`feeds`.

Sample::

    Application('molly.apps.feeds.news', 'news', 'News'),
