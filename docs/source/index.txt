The Molly Project
=================

The Molly Project is a framework written in Django and Python for developing
a location-aware mobile portal with particular emphasis on Higher Education.

It is released under the :doc:`Academic Public License <license>`, and is
available from `mollyproject.org <http://mollyproject.org/>`_.
